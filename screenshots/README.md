# 截图占位说明

本目录用于存放项目截图，请按照以下要求准备图片：

## 需要的截图

### 1. banner.png
- **用途**: README 顶部的项目横幅图
- **建议尺寸**: 1280x640 像素
- **内容建议**: 展示应用运行效果或 Logo

### 2. statusbar.png
- **用途**: 展示状态栏中的价格显示效果
- **建议尺寸**: 800x100 像素
- **内容建议**: 截取 macOS 顶部状态栏，显示 BTC 价格和迷你图表

### 3. detail.png
- **用途**: 展示行情详情窗口
- **建议尺寸**: 800x1000 像素
- **内容建议**: 点击状态栏图标后弹出的详情窗口，包含价格、涨跌幅等信息

## 截图技巧

1. **macOS 截图快捷键**:
   - `Cmd + Shift + 3`: 全屏截图
   - `Cmd + Shift + 4`: 区域截图
   - `Cmd + Shift + 4`, 然后按 `空格`: 窗口截图

2. **图片格式**: PNG（推荐）或 JPG

3. **图片优化**: 
   - 保持清晰度
   - 文件大小建议 < 1MB
   - 去除敏感信息（如个人数据）

## 添加截图后

将截图放入此目录后，README.md 中的图片链接会自动生效。
